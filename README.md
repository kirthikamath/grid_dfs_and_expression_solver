# grid_dfs_and_expression_solver
## Problem 1: Operator Combinations
- Reads the file containing equations in the format described.
- For each line parse the target number and the list of operands
- Genrated all possible combinations of + and * operators
- Evaluates expressions from left to right and matches the target.
- Prints the final sum after processing all the lines in the input files.

## Problem 2: Trailhead Score(DFS)
- Finds all trailheads(0s) in grid
- Uses DFS with constraint(next = current + 1)
- Counts unique reachable peaks(9s)
- Calculates the total score

## Concepts Used
- Arrays
- File Handling
- DFS  (Depth First Search)
- Recursion
